# Gerador de Senhas Seguras

Este projeto é um gerador de senhas seguras em Python feito como um estudo. Permite ao usuário definir o comprimento da senha e utiliza letras maiúsculas, minúsculas, números e símbolos para criar senhas fortes.

## Como usar

Execute o script `password_generator.py` informando o comprimento desejado da senha:

```bash
python password_generator.py --length 12
```

## Funcionalidades
- Geração de senhas aleatórias
- Parâmetro para definir o comprimento
- Utiliza letras, números e símbolos

## Dependências
- Python 3.x
- (Opcional) argparse
